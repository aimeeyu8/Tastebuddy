# TasteBuddy: LLM Restaurant Assistant

TasteBuddy is a Large Language Model (LLM)-powered restaurant assistant that aims to solve the issue of picking restaurants in group settings. Unlike other solutions that use filtering, TasteBuddy hopes to understand multiple users’ chats and extracts preferencesto suggests suitable, allergy-safe dining options in New York City.
